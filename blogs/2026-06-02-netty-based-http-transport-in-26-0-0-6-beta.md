Open Liberty 26.0.0.6-beta introduces a preview of a Netty-based HTTP transport that modernizes the underlying foundation for HTTP/1.1, HTTP/2, WebSocket, JMS, and SIP communications with zero migration required. The implementation includes a fix for a previously reported 100-continue issue and existing applications and configurations continue to work unchanged. Developers are invited to test real-world scenarios including HTTP transfers, WebSocket communications, timeout handling, and JMS messaging before general availability.
